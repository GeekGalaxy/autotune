What is Autotune?

Autotune tunes your machine's Linux kernel to give you the best performance for the service you are trying to run. This is done by taking account of the server profile of your machine (e.g. Go, Nodejs, Apache) and tuning the machine with the correct settings.

Install

Installation is easy. Specify the server profile of your machine and run the curl command.
